## About This Project

A luxury brand website for a fictional Assam Tea company — built as a portfolio showcase by DigiRise India. Features include:

- **Video frame scrubbing** — 72MB mp4 extracted into ~300 frames, scrubbed via scroll
- **GSAP ScrollTrigger animations** — 10 cinematic text overlays synced to scroll position
- **Custom cursor** with magnetic ring effect
- **Product collection** with 3 tea blends (DAWN, DUSK, NIGHT)
- **Story stats strip**, newsletter section, and premium footer
- **Floating WhatsApp CTA** — direct lead generation for DigiRise India
- **DigiRise India credit badge** in footer (links to WhatsApp)

---

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- GSAP 3 + ScrollTrigger
- Express.js (for local serving)
- Google Fonts: Cormorant Garamond + Inter

## Run Locally

```bash
npm install
npm start
# Open http://localhost:3000
```

## GitHub Pages Deployment

Since this site uses a video file, for GitHub Pages:

1. Push all files including `public/tea-story-final.mp4`
2. Set GitHub Pages source to `main` branch → `/public` folder
3. Or use the root with a custom `index.html` redirect
